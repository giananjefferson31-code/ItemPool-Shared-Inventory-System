# ItemPool-Shared-Inventory-System

The system is a multi-user inventory management application where users can register and log in to manage shared items within a common item pool. Users can add items to the inventory pool, making them visible and accessible to other users. Any user can claim or “get” an available item, which temporarily grays out it from the public pool and assigns it to their account. Users may also return claimed items back to the shared pool for others to access. The system includes role-based item ownership where only the original uploader of an item has permission to edit or delete that item. Other users are limited to viewing, claiming, and returning items. All item transactions and updates are synchronized through the database to maintain real-time inventory availability and user accountability.,..
